# Additional Settings

> **Version:** v0.0.2 
> **For Geometry Dash:** 2.2081 (Win/Android) 
> **Author:** [retre1](https://github.com/retrei1)

---

##  Description

The mod adds additional settings that are missing from the standard Geometry Dash:

- **FPS Bypass** — removes the 60 FPS limit
- **Layout Mode** — edit levels directly during testing
- **Show FPS** — displays the frame counter on the screen
